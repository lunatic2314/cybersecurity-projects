# Endpoint Security

This folder focuses on securing and monitoring endpoint devices through hands-on labs and practical security exercises.

## Topics Covered
- Endpoint protection concepts  
- Malware detection and prevention  
- System hardening and patch management  
- Log collection and event analysis  
- Incident detection and response basics  

## Skills Practiced
- Endpoint threat identification  
- Security configuration and hardening  
- Troubleshooting security issues  
- Documentation and reporting  

## Goal
To strengthen endpoint security skills applicable to SOC, IT security, and entry-level cybersecurity roles by applying defensive strategies used in real-world environments.
