network-defense # Network Defense Labs

This folder contains hands-on labs focused on network security concepts, traffic analysis, and defensive techniques used to detect and mitigate threats.

## Topics Covered
- Network traffic analysis
- Firewall and IDS/IPS concepts
- Identifying suspicious connections
- Basic threat detection techniques

## Tools Used
- Wireshark
- Network logs
- Command-line networking tools

## Goal
To build foundational skills in monitoring, analyzing, and defending network environments.
