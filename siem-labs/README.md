# 🚨 Wazuh + Suricata SIEM & IDS Integration Lab

# 🔎 Project Overview
This project demonstrates the design and deployment of a security monitoring lab integrating:

-Wazuh (SIEM/XDR platform)

-Suricata (Network IDS/IPS)

-OpenSearch (Data indexing engine)

-Ubuntu Server 24.04

-Virtualized lab environment

The goal was to simulate enterprise SOC monitoring capabilities by ingesting network intrusion alerts from Suricata into Wazuh for centralized analysis and visualization.
